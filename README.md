# Manual Testing Project – EverShop.io

## Overview

This project demonstrates a complete **Manual Testing** workflow on the EverShop demo application. The assessment covers requirement analysis, test case design, manual test execution, API testing using Postman, UI testing, and defect reporting.

**Application Under Test:** https://demo.evershop.io/

---

# Assessment Objectives

- Requirement Analysis
- Test Case Design
- Manual Test Execution
- API Testing (Postman)
- UI Testing
